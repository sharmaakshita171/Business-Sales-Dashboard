# Business-Sales-Dashboard

# Overview
This interactive Power BI dashboard analyzes business sales performance using the Superstore dataset. It provides key insights into **revenue, profit, and customer trends** across *regions, categories, and time*.

# Objectives
► Track total sales, profit, quantity, and profit margin  
► Identify top-performing regions and product categories
► Analyze monthly sales trends and customer segmentation
► Build dynamic visuals for actionable business insights

# Tools
Power BI: Data modeling, DAX measures, visual design
Excel: Dataset review
Dataset: [Superstore Dataset from Kaggle] (https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)

# Dashboard Features

► Interactive filters by **Region, Category, and Customer Segment**
► Dynamic KPI cards and responsive visuals
► Geographic sales map and Top 10 customer analysis

# Insights

► **West region** generated the highest sales but moderate profit
► Technology category contributes most to revenue
► Seasonal sales spike observed in **October–December**
► **Top 10 customers** significantly impact total revenue
